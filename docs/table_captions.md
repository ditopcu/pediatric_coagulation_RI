## Publication Tables — Main Text

### PUB Table 1 — PUB_table_1_study_population.csv
**Title:** Study population characteristics by coagulation test.
**Caption:** Number of observations before and after 3SD outlier exclusion, sex distribution, and age summary statistics for PT, aPTT, and fibrinogen. ICU patients excluded at data extraction. Roche Cobas t511/t711, tertiary pediatric hospital, Turkey, 2018–2024.

### PUB Table 2 — PUB_table_2_reference_intervals.csv
**Title:** Reference intervals by estimation method and age group.
**Caption:** Reference intervals (2.5th–97.5th percentile) estimated by indirect (refineR), direct (a posteriori), continuous (GAMLSS), and manufacturer-supplied methods for PT (seconds), aPTT (seconds), and fibrinogen (g/L). 90% confidence intervals for lower and upper reference limits. Continuous GAMLSS values represent age-averaged estimates. For aPTT, separate intervals for 1–12 y and 12–18 y based on Harris-Boyd partitioning (z = 18.2, Table 6).

### PUB Table 3 — PUB_table_3_false_positive_rates.csv
**Title:** False-positive rates by reference interval source with 95% bootstrap confidence intervals.
**Caption:** Proportion of 3SD-filtered results classified as outside reference limits for each RI source. Bootstrap 95% CIs based on 1000 iterations. Expected false-positive rate for 2.5th–97.5th percentile RI is 5%. Guven et al. RIs applied to PT and aPTT only (no fibrinogen data in that study).

### PUB Table 4 — PUB_table_4_guven_transferability.csv
**Title:** Transferability assessment of Guven et al. reference intervals.
**Caption:** CLSI EP28-A3c 20-sample bootstrap validation of Guven et al. (2026) reference intervals applied to our population at each integer age. Pass criterion: ≤2/20 samples (10%) outside RI at ≥90% of 200 bootstrap iterations. Neither PT nor aPTT RIs from Guven et al. were transferable to our population at any age.

### PUB Table 5 — PUB_table_5_international_comparison.csv
**Title:** International comparison of pediatric coagulation reference intervals.
**Caption:** Reference intervals for PT, aPTT, and fibrinogen from published studies using different analyzers, populations, and estimation methods. All values represent 2.5th–97.5th percentile limits unless otherwise noted. PT values are analyzer-dependent; direct comparison across platforms requires caution.

### PUB Table 6 — PUB_table_6_harris_boyd.csv
**Title:** Harris-Boyd test for age partitioning at 12 years.
**Caption:** Statistical justification for age-specific reference intervals using the Harris-Boyd z-score criterion (threshold z > 3). For aPTT, the difference between children <12 years and ≥12 years was clinically and statistically significant (z = 18.2), supporting age partitioning. For PT, z = 5.0 exceeded the threshold but the mean difference (0.06 s) was clinically negligible; a single PT interval was retained.

## Publication Tables — Supplementary

### PUB Table S1 — PUB_table_S1_descriptive_all.csv
**Title:** Descriptive statistics for all coagulation tests by integer age.
**Caption:** Combined descriptive statistics for PT (seconds), aPTT (seconds), and fibrinogen (g/L) at each integer age after 3SD outlier exclusion. Includes mean, SD, median, IQR (Q25–Q75), range (min–max), skewness, and kurtosis.

### PUB Table S3 — PUB_table_S3_continuous_RI_values.csv
**Title:** Age-specific continuous reference interval values from GAMLSS modeling.
**Caption:** Numerical values of the 2.5th, 50th (median), and 97.5th percentile continuous reference interval curves with 90% bootstrap confidence intervals for PT (seconds), aPTT (seconds), and fibrinogen (g/L) at each integer age (1–18 years). GAMLSS BCCG family with power transformation; 5 bootstrap iterations.
